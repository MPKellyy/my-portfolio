ITN-SelfSearch

Description

The goal of ITN SelfSearch is to help customers find where items are in a real-world store. This software calculates the most optimal route toward a product and displays a visual map path for customers to use as a reference. Software also features an administrative mode that allows for product database importation and map creation.
